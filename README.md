# 🛒 Shopping List App

브라우저 로컬에 저장되는 가볍고 이쁘은 **쇼핑 리스트 웹앱**입니다. 단일 HTML 파일로 동작하며, 동공된 정적 서버(`server.js`)를 통해 로컬에서 손쉽게 실행할 수 있습니다.

---

## ✨ 주요 기능

- ➕ **아이템 추가** — 입력후 "추가" 버튼 또는 `Enter` 키로 등록
- ✅ **체크하기 / 해제** — 구매 완료 항목 표시 (취소선 효과)
- ✖️ **개별 삭제** — 항목 오른쪽 ✕ 버튼으로 제거
- 🧹 **완료 항목 일괄 비우기** — 체크된 항목만 한 번에 삭제
- 📊 **진행 상황 표시** — 총 항목 수 · 남은 항목 수
- 💾 **자동 저장** — `localStorage`에 보관되어 새로고침/재방문 시에도 유지
- 📱 **반응형 UI** — 모바일/태블릿/데스크톱 대응

---

## 🚀 빠른 시작

### 1. 단일 HTML로 열기 (가장 간단)

`index.html`을 브라우저에 직접 드래그 하거나 더블클릭하면 증시 실행됩니다. 별도의 빌드 과정은 필요 없습니다.

### 2. 로컬 정적 서버로 실행

Node.js가 설치되어 있으면 포함된 경량 서버를 사용할 수 있습니다:

```bash
node server.js
```

그다음 브라우저에서 아래 주소로 접속:

```
http://localhost:8765/
```

포트를 바꾸려면 `server.js` 상단의 `PORT` 상수를 수정하세요.

## 🔧 기술 스택

- **HTML5 / CSS3 / Vanilla JavaScript** (프레임워크 없음)
- **Web Storage API** (`localStorage`) — 웹 표준 키-값 저장소
- **`crypto.randomUUID()`** — 항목 고유 ID 생성
- **Node.js `http` 모듈** — 외부 의존성 없는 정적 서버

---

## 💾 데이터 저장 방식

리스트는 브라우저의 `localStorage`에 `shopping-list-items` 키로 JSON 형태로 저장됩니다:

```json
[
  { "id": "<uuid>", "text": "우유", "checked": false },
  { "id": "<uuid>", "text": "계란", "checked": true }
]
```

> ⚠️ `localStorage`는 브라우저/기기별로 분리되어 있으며, 브라우저 데이터를 삭제하면 리스트도 함께 사라집니다.
